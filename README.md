# ✨ Star-Wars-Database ✨

This project was created as part of a semester assignment and includes the following steps:

- 🧠 **Database Design**  
  Requirements analysis, description and the initial design of the database structure.

- 📊 **Conceptual Schema Modeling**  
  Creation of an ER diagram describing entities, their attributes, and relationships.

- 🛠️ **Implementation in PostgreSQL**  
  Translating the conceptual model into a real database using PostgreSQL.

- 🔍 **Query Implementation using SQL**  
  Writing and testing SQL queries to retrieve the required data.

- 🎤 **Presentation and Project Defense**  
  Summarizing the results and demonstrating the functional database and queries during the final presentation.

> 💡 The project is hosted on the [dbs.fit.cvut.cz](https://dbs.fit.cvut.cz) portal.  
> Access is restricted to students of the Faculty of Information Technology at CTU in Prague.
